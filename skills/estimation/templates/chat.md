# Template: Chat output

Present the estimate directly in the conversation as formatted markdown. No files are created or modified.

## When to use

- Quick sizing discussions
- Estimating verbal descriptions, Jira tickets, or other non-file inputs
- Projects that do not track estimates in files

## Example

```markdown
## Estimate
**Points:** 3

Touches the search component and results page with a clear approach; minor uncertainty around multi-language placeholder behavior.

| Factor | Score |
|--------|-------|
| Complexity | Medium |
| Uncertainty | Low |
| Effort | Medium |
| Risk | Low |
| Dependencies | Low |
```
