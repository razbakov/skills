# Caption Patterns — per-cell shapes

For each cell type, a proven caption shape. Adjust voice to the brand.

## A1 — Hero / Festival announcement (the first impression)

**Shape:** assertion → 4-line bullet of essentials → CTA → tagline

```
<Brand> 2027 — the festival is real.

🇨🇺 <N> days · <dates> · <venue>
🎤 <Headliner + supporting acts>
💃 <signature feature, e.g. "14 workshop artists · Roots vs. Revolution">
🎟 <Offer + urgency, e.g. "100 super-early-bird tickets €50 — drop Saturday">

Sign up at <url> → join the <community channel> when you're done. Be loud about who you want on stage.

<tagline>
```

Length: 100-150 words. The CTA is mandatory.

## A2 — Headliner / second-act reveal

**Shape:** name as opening punch → context → date+venue → ticket urgency → handle

```
<HEADLINER>. <FESTIVAL OR LIVE IN [CITY]>.

<Day, date>. <Short paragraph naming what makes them legendary in 1-2 sentences>.

<Specific ticket offer with date>.

<url> · <@handle>
```

Length: 60-100 words. The name punch at the top is non-negotiable — it's the hook.

## A3 — Tagline card

**Shape:** tagline as caption opener → one paragraph of context → CTA

```
<Tagline as the first 3 lines, broken naturally>

<One paragraph: who this is for, how often it happens, what makes it different>.

<url> → <next step, usually a community channel>.
```

Length: 50-80 words. The caption should echo the on-image text without quoting it verbatim — paraphrase.

## B1 — Roots / heritage / tradition photo

**Shape:** single word punch → paragraph honoring the tradition → connection to the schedule

```
ROOTS.

<2-3 sentences honoring the tradition, the elders, the source of the music/genre/practice>.

<When this shows up in the schedule, e.g. "Thursday opens with the elders. The week begins where the music began.">

<dates> · <url>
```

Length: 50-80 words.

## B2 — Dates card / practical info

**Shape:** "save the four days" or similar imperative → 3-line bullet with date/venue/headliner → what the ticket includes → CTA

```
Save the <N> days.

🗓 <dates>
📍 <venue>
🇨🇺 <headliner anchor>

<one line on what the experience includes>.

<url> · <@handle>
```

Length: 40-60 words. Practical, scannable, light on adjectives.

## B3 — Revolution / new generation / innovation photo

**Shape:** single word punch → paragraph framing the discovery thesis → connection to the schedule

```
REVOLUTION.

<2-3 sentences framing why the new/emerging side matters. What the festival/brand finds before others do>.

<dates> · <url>
```

Length: 50-80 words. Less ornate than B1; this side is "look forward".

## C1 — Concept card (the dual / the idea)

**Shape:** the concept name → the explanation in one sentence → tactile detail → repeat structure

```
<CONCEPT NAME>.

<One sentence explaining the dual/the idea: "The orchestras that built the tradition meet the teachers writing its next chapter.">

<Tactile detail line that grounds it: "Father and son. Past and present. Same stage. Four days.">

<dates> · <url>
```

Length: 50-70 words.

## C2 — Mission card (the boldest claim)

**Shape:** "this is the thing we are loud about" → the bold claim → bilingual version (optional) → date + url

```
This is the thing we are loud about:

<The mission, often as a complete sentence with one accent word that matched the image>

<If multilingual: same line in the secondary language>

<dates> · <venue>

<url>
```

Length: 50-80 words. The bilingual hook is high-leverage if your audience is multilingual.

## C3 — Wordmark / brand mark (the foundation)

**Shape:** brand name as opening → tagline → essentials → handle CTA

```
<BRAND NAME>.

<Tagline>

<one line of essentials: dates · venue · headliner>

Link in bio · <bilingual "join us" hooks if multilingual> → <url>
```

Length: 30-50 words. This is the most foundational caption because it'll remain at the bottom-right of the grid forever. Make it tight and time-invariant (no specific drop dates here).

## Hashtag patterns

Per post, 8-15 tags arranged as:

```
#BrandTag #BrandTag2 #BrandTag3
#NicheSpecific #NicheSpecific2 #NicheSpecific3 #NicheSpecific4
#Location #Location2 #LocationLanguageVariant
#OneOrTwoBroader
```

Examples for a Cuban festival in Munich:
- Brand: `#AguaPichi #MunichsCubanFestival #RootsAndRevolution`
- Niche: `#SalsaCubana #TimbaCubana #AfroCuban #BachataDominicana`
- Location: `#Munich #München #MunichEvents #MunichDance`
- Broader: `#CubanMusic #LatinFestival`

The `.\n.\n.\n` "hidden hashtag block" pattern is dated. Modern best practice is to integrate hashtags into the caption tail naturally (one line at the bottom).

## Tone notes

- **Captions are not poster text.** The image already says it visually. The caption adds context, voice, CTA — not a repeat.
- **One emoji per post maximum** unless the brand voice is explicitly playful. Most festival/business launches use 0-1 emoji.
- **No "we are excited to announce" openings.** Use punch openings: a name, a date, a single word in all caps.
- **Always end with a CTA.** Link in bio, join the WhatsApp group, tickets at the URL. Never let a caption end on a mood.
