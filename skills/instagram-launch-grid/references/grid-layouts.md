# Grid Layouts — proven 3-row narrative shapes

A 9-post grid (3×3) reads as a single curated installation. The 3 rows give you a story arc.

The user picks one narrative shape during input gathering; you then map their content to each cell.

## Layout 1 — Hook / Substance / Brand (default for festivals + events)

Top row hooks the visitor ("what is this and why now"). Middle row delivers the substance ("here are the facts and the people"). Bottom row anchors the brand ("here's who we are").

```
[A1 Hero flyer]      [A2 Headliner reveal] [A3 Tagline card]
[B1 Roots photo]     [B2 Dates / venue]    [B3 Revolution photo]
[C1 Concept card]    [C2 Mission card]     [C3 Wordmark]
```

Best for: festivals, single events, conferences, product launches with a date.

## Layout 2 — People / Place / Plan

Top row puts the humans first ("these are the people"). Middle row anchors location and atmosphere. Bottom row shows the plan or schedule.

```
[A1 Founder photo]   [A2 Team group shot]  [A3 Customer photo]
[B1 Place hero]      [B2 Place detail]     [B3 Place atmosphere]
[C1 Plan / steps]    [C2 Schedule / dates] [C3 Wordmark]
```

Best for: new community, club, retreat, restaurant, studio launch — anywhere "who runs this" is the conversion lever.

## Layout 3 — About / Offer / Proof

Top row says what it is. Middle row explains the offer. Bottom row shows proof / credibility.

```
[A1 Hero shot]       [A2 Tagline / promise] [A3 Founder note]
[B1 Offer #1]        [B2 Offer #2]          [B3 Offer #3]
[C1 Testimonial]     [C2 Numbers / proof]   [C3 Wordmark]
```

Best for: product launch with a clear offer, service business, course / cohort launch.

## Layout 4 — Past / Present / Future

A "this is who we've been, who we are, where we're going" frame. More narrative-heavy.

```
[A1 Origin story]    [A2 First version]    [A3 Founder note]
[B1 Today photo]     [B2 Current offering] [B3 Customer impact]
[C1 What's coming]   [C2 Vision / future]  [C3 Wordmark]
```

Best for: rebrands, 5-year anniversaries, mature brands relaunching, founder-led brands.

## Layout 5 — Theme triptych (all rows same theme, different angle)

Three rows × three angles of one core idea.

```
[A1 angle 1]         [A2 angle 1]          [A3 angle 1]
[B1 angle 2]         [B2 angle 2]          [B3 angle 2]
[C1 angle 3]         [C2 angle 3]          [C3 angle 3]
```

Best for: art collective, fashion brand, magazine, gallery — when the visual identity is more important than narrative.

## How to pick

| Signal | Layout |
|---|---|
| Has dates + headliner | Layout 1 |
| Has a face / founder / team is the story | Layout 2 |
| Has a sales offer to convert | Layout 3 |
| Has a history to tell | Layout 4 |
| Has a strong visual style and wants to flex it | Layout 5 |

When in doubt, default to Layout 1 — it works for most festivals/events/launches.

## Cell job rules (apply across all layouts)

- **A1 (top-left)** = first impression. Always the strongest single image. Hero.
- **A2 (top-middle)** = the "and also" reveal. Headliner, second feature, surprise.
- **A3 (top-right)** = the brand voice in one phrase. Type-led tagline card.
- **B1, B3** = emotional anchor photos. Real humans, real moments, photographic. NOT text cards.
- **B2** = the practical card. Dates, location, "save the date", offer details.
- **C1** = the concept / why we exist. Type-led with one strong sentence.
- **C2** = the mission / boldest claim. Type-led on contrasting background (often dark).
- **C3 (bottom-right)** = the brand mark. Logo / wordmark. Time-invariant. Never anything time-bound here.

## Visual rhythm checklist

Before generating, verify the planned grid:

- [ ] At least 2 cells are real-people photographs (not text cards)
- [ ] No row is 3 text cards in a row (boring brochure look)
- [ ] No row is 3 photos in a row (chaotic)
- [ ] One accent color appears in at least 4 of 9 cells (visual unity)
- [ ] Bottom-right (C3) is time-invariant brand content
- [ ] Top-left (A1) is the single strongest visual you have
